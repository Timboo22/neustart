## ✨ Über das Projekt
**neustart** ist der Ort, an dem Ideen zu Code werden. Dieses Repository beherbergt das Frontend (und mehr), um eine performante und benutzerfreundliche Web-Erfahrung zu schaffen. 

Warum der Name? Weil jedes großartige Projekt mit einem mutigen **neustart** beginnt.
